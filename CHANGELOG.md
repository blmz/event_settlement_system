# 更新日志

所有重要的项目更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-02-18

### 新增
- 核心事件结算系统
  - EventData 事件数据类
  - EventModifier 修改器系统
  - EventSettlementManager 单例管理器
  
- 修改器类型支持
  - 加法修改器 (ADD)
  - 乘法修改器 (MULTIPLY)
  - 覆盖修改器 (OVERRIDE)
  - 自定义修改器 (CUSTOM)
  
- 条件系统
  - 事件类型条件
  - 标签检查条件
  - 数值范围条件
  - 随机触发条件
  - 额外数据条件
  - 组合条件（与/或）
  
- 多阶段处理
  - 事件前置阶段 (before_process)
  - 事件处理阶段 (processing)
  - 事件后置阶段 (after_process)
  - 事件结算完成 (settled)
  
- 管线系统
  - SettlementPipeline 管线类
  - PipelineStage 阶段类
  - 多阶段顺序执行
  - 可中断执行流程
  
- 工具类
  - EventUtils 辅助函数集
  - 快速创建常见事件
  - 内置条件函数生成器
  - 调试打印工具
  
- 文档系统
  - 完整的使用文档
  - API 快速参考
  - README 使用指南
  - 代码内联文档
  
- 示例系统
  - 示例场景 (example_usage.tscn)
  - 完整的演示代码
  - 多种使用场景展示

### 特性
- 高性能修改器排序
- 完整的历史记录追踪
- 丰富的信号系统
- 链式调用支持
- 灵活的条件判断
- 零外部依赖
- 全中文注释和文档

### 技术细节
- 使用 Godot 4.6+ class_name 系统
- 完整的类型提示

- 单例自动注册

## 版本说明

### 版本号格式
遵循语义化版本 `主版本号.次版本号.修订号`：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正

### 变更类型
- **新增 (Added)**：新功能
- **变更 (Changed)**：现有功能的变更
- **弃用 (Deprecated)**：即将移除的功能
- **移除 (Removed)**：已移除的功能
- **修复 (Fixed)**：Bug 修复
- **安全 (Security)**：安全性修复
